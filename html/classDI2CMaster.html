DI2CMaster::DI2CMaster ( DI2CBusHandle  i2cBusHandle,
uint16_t  i2cMaxBufferLenght = 0 
)

Construct a new DI2CMaster::DI2CMaster object.

Parameters
i2cBusHandle-> an handle for the i2c bus (can obtained by DI2CBus() class).
i2cMaxBufferLenght-> max lenght of tx buffer used in each transaction.
  • If value is 0, no limit are set.
  • If value is greater than 0, write...() and askFor...() API truncate buffer of each transaction to this value, send...() API do as many transaction as need to send all data.
