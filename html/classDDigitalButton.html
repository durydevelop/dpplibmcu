Release STATE: button is not in StatePressed state.

Press STATE: button is in StatePressed state.

LONG_PRESSED 

Pressed TRIGGER: press-release after PressedMillis time.

LONG_PRESSING 

Long pressed TRIGGER: button is kept pressed for longer than LongPressedMillis.

DBL_PRESSED 

Long Pressing STATE: after LONG_PRESSED, if button still pressed, the state change in LONG_PRESSING until button is released.

DBL_PRESSING 

Double pressed TRIGGER: press-release-press within DblPressSpeedMillis time.

RELEASED 

Double pressing STATE: after DBL_PRESSED, if button still pressed, the state change in DBL_PRESSING until button is released.

Released TRIGGER: release after PRESSED, LONG_PRESSED, DBL_PRESSED.
