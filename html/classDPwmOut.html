◆ setMicros() [1/2]

bool DPwmOut::setMicros ( uint16_t  us)

Set duration of pwm pulse in microseconds.

If pulse is greater than current period, duty cycle wil be set 100%. The frequency is 1000000 / (pulseOn us + pulseOff us) Hz, so: pulseOff us = (1000000 / frequency hz) - pulseOn us

The set duty cycle will be pulseOn / (pulseOn + pulseOff) * 100 %

us
